
# Chatbot

An AI chatbot API + Fronend with Tensorflow + flask




## Installation

Install These python Libraries (Im using Anaconda)

```bash
 conda install -c anaconda flask-cors
```
```bash
 conda install tensorflow
```
```bash
 conda install flask

```
